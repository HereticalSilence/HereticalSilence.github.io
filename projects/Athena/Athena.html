Athena Discord Bot

Written using Python and Discord.py

Athena originally began as a bot called Maximus back in 2018. I began developing the bot as a side project to further develop my Python skills and particularly OOP during my GCSE. Since then the feature set has grown alongside my skills. I greatly attribute this bot to my programming and developmental success.

  • PostgreSQL database
  • YouTube Audio Streaming
  • Chat Filtering and Logging
  • Automod Functionality
  • Moderation commands
  • Custom commands
  • Fun commands
  • Utility commands
